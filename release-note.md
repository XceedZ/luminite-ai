# Release Notes

## Version 1.3.1
_latest_

- Added prompt enhancement feature in App Builder - improve and refine prompts with AI assistance.
- Added file tree view in code tab showing imported files and components.
- Improved code tab with resizable panels for better code navigation.
- Added basic icon library (lucide-react) available globally in preview environment.
- Enhanced icon conflict prevention - automatically removes duplicate icon declarations from generated code.
- Improved error handling for HMR (Hot Module Replacement) issues.

## Version 1.3.0
_Recently updated_

- Added billing page with plan usage bars and payment method.
- New pricing page with Free, Pro and Enterprise plans.
- Improved tabs styling and icon support across settings.
- Dynamic breadcrumbs no longer prepend Dashboard incorrectly.

## Version 1.2.9
- Context selector added in quick create input.
- File uploads now show preview with remove action.
- Type safety improvements for AI icon map and messages.
- Fixed build error from cookies API and model icons.

## Version 1.2.8
- Improved scrolling and layout stability in chat pages.
- Accessibility tweaks for buttons and menu items.
- Minor UI polish on dropdowns and separators.
- General performance and reliability fixes.
